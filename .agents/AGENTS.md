# プロジェクト行動ルール (AGENTS.md)

本プロジェクトにおいて、AI アシスタント（Antigravity）が従うべき絶対的な行動ルールです。

## 1. 日本語による表示
* 開発プロセス、ログ、応答、確認メッセージはすべて日本語で表示してください。

## 2. 計画承認の原則
* 実装や変更（リファクタリング、バグ修正含む）を行う前に、必ず具体的な【実装プラン】をユーザーに提示し、合意（承認）を得てから手を動かしてください。
* ユーザーがプラン提示のみを求めた場合は、実装を行ってはいけません。

## 3. GitHub への自動プッシュ禁止（最重要）
* ユーザーの明示的な許可・指示を得る前に、GitHub などのリモートリポジトリへ勝手にプッシュ（`git push`）することを固く禁じます。
* 実装完了後、コミットを作成した段階で一度ユーザーに確認し、「GitHubにプッシュしてください」という明示的な許可を得てからプッシュコマンドを実行してください。

## 4. アイコンは線画（アウトライン）に統一する
* 絵文字（例：➕, 🗑️, ✏️, 🔄, 📅, 📊）をUI上のアイコンとしてそのまま使用せず、シンプルな線画（アウトライン）のアイコン（SVGコード埋め込みなど）に統一してください。
